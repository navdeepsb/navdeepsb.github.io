About

HomeArte is a home decor website specializing in modular kitchens. I designed and developed it's front-end.

Home

I decided to go for a spacious home page that highlights the photos in the page. A carousel on the top presents to the user various designs of kitchens. Each photo in the catalogue section reveals a gallery of related photos that can be browsed easily.

Login

Below design shows the website’s login page with the sidebar revealed. This sidebar is accessible on every page and provides a quick way to navigate around the website. As you can see, when the sidebar is revealed, the other section is dimmed out so the user can focus on the sidebar. A sidebar is the perfect way to abstract a navigation bar – there is a single link to show or hide the sidebar and a lot of links can be accommodated here. Plus, it saves time required to redesign a navigation bar for handheld devices.

Conclusion

HomeArte was my freelancing project that I justified properly. The website has a soothing aura that appeals to all.
