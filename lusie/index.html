Lusie

Personal project, Developer

About

Lusie is a casual housie/lotto/tambola web app written in Vanilla JavaScript. The highlight of this project is not the idea but its execution. I developed this app solely to put prototypal JavaScript in use. The source code is proudly hosted on Github, check it out.

Screenshots

Below is a ridiculously simple layout for the app. Gigantic number depicts the current number drawn and the circled numbers are the ones already drawn. Two handy buttons to start/end and pause/resume the gameplay. The text box besides the buttons contains the seconds after which each number should be drawn (absolutely configurable, just pause the game, change the value and resume to reflect the changes).


And a dark theme for the night watchers.

Conclusion

I am not lying when I say that I use this app for housie sessions with my family. Totally relieves the person drawing the numbers so he/she can also play.
